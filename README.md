# Juice

Juice is the power management system for [The Flip](https://www.theflip.museum/).

Some features:
* Turns machines off and on in bulk or individually
* Tracks power usage over time
* Where possible, measures attract vs play vs idle modes
* Tracks machine usage over time
* Watches for unusally high, sustained power draw and shuts machines down automatically to prevent damage
* Integrates with our maintance tracking system, [FlipFix](https://github.com/The-Flip/juice) to fetch machine information and to post problem reports
